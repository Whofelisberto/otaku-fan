# Anime Fan

## - About

### - The project is a creation of my mind, I love animes and one day I want to make a web application about animes, where everyone can use it.

## Aplication Web - https://otaku-fan.vercel.app/

# Technologies used in the project

## - React , Javascript and Tailwind

# - Summary

## - this project was challenging I loved doing it, I love animes. I thought practically about everything before doing it and how to carry out this project, technologies to use, etc. I put all the information in JSON because that way the project would be cleaner and it would give me a more practical way to use them, thus keeping the layout light without being too cluttered with images.
